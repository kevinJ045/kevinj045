# Lulu

A **magical Lua runtime** that bundles code, resolves dependencies, and builds final executables.

## Usage

<div className="grid grid-cols-1 md:grid-cols-2 gap-8">

<Card>
  <CardHeader>
    <CardTitle>Run a Single File</CardTitle>
  </CardHeader>
  <CardContent>
    <CodeBlock language="bash" content="lulu run single.lua" />
  </CardContent>
</Card>

<Card>
  <CardHeader>
    <CardTitle>Run a Project</CardTitle>
  </CardHeader>
  <CardContent>
    <CodeBlock language="bash" content="lulu run main.lua" />
  </CardContent>
</Card>

<Card>
  <CardHeader>
    <CardTitle>Bundle Project</CardTitle>
  </CardHeader>
  <CardContent>
    <CodeBlock language="bash" content="lulu bundle main.lua /path/to/main.lulib" />
  </CardContent>
</Card>

<Card>
  <CardHeader>
    <CardTitle>Build Executable</CardTitle>
  </CardHeader>
  <CardContent>
    <CodeBlock language="bash" content="lulu build /path/to/project
    # or
    lulu build # means lulu build ." />
  </CardContent>
</Card>

</div>

## Dependencies

<Card>
  <CardHeader>
    <CardTitle>Resolve Dependencies</CardTitle>
    <CardDescription>Fetch and cache dependencies from URLs or Git repositories.</CardDescription>
  </CardHeader>
  <CardContent>
<CodeBlock language="bash" content="lulu resolve .
lulu resolve github:user/repo" />
  </CardContent>
</Card>

<Card>
  <CardHeader>
    <CardTitle>Manage Cache</CardTitle>
    <CardContent>
  <CodeBlock language="bash" content="# List all cached packages
lulu cache list

# Clear cache
lulu cache clear

# Remove a specific package
lulu cache remove <package_url>" />
    </CardContent>
  </CardHeader>
</Card>
